## Commands

Here's a list of all the commands (brackets denote optional parameters).

### 8ball

Ask lumina a question.

Aliases: `8` `8b`

`l!8ball tu aimes le pamplemousse ?`

### avatar

View the avatar of the tagged user, or your own.

Aliases: `av` `dp` `icon` `pfp`

`l!avatar [@user]`
